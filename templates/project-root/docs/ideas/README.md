# Ideas

Exploratory architecture discussions and design artifacts. Ideas are the first stage of the governance pipeline.

## Workflow

1. Explore via `/new-idea`
2. Review via `/review-idea-doc`
3. If contract changes needed: progress to `/new-plan`
4. If implementation-only: progress to `/new-sprint`

## Naming convention

Files use sequential numbering: `IDEA-NNN-YYYY-MM-DD-<topic>.md` (e.g., `IDEA-001-2026-03-13-user-authentication.md`). The `/new-idea` skill assigns the next available number automatically.
